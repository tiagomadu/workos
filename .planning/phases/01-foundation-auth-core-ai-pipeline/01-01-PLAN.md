**frontend/src/components/auth/sign-in-button.tsx** — Client component (`"use client"`). Uses browser Supabase client. Calls `supabase.auth.signInWithOAuth({ provider: "google", options: { redirectTo: origin + "/auth/callback", queryParams: { access_type: "offline", prompt: "consent" } } })`. CRITICAL: `access_type: "offline"` and `prompt: "consent"` ensure a refresh token is issued. Renders a shadcn Button with Google icon.

**frontend/src/components/auth/sign-out-button.tsx** — Client component. Calls `supabase.auth.signOut()` then `router.push("/login")` using `useRouter`. Renders a shadcn Button variant="ghost".
