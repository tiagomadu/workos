**backend/app/services/owner_resolution.py** -- Fuzzy matching service:

1. `def compute_similarity(name1: str, name2: str) -> float` -- Case-insensitive fuzzy match. Use `difflib.SequenceMatcher` from stdlib (no extra dependency). Compare lowercase stripped strings. Return ratio (0.0 to 1.0).

2. `def check_alias_match(owner_name: str, person_aliases: str) -> bool` -- Split aliases by comma, strip whitespace, check if any alias case-insensitively equals the owner_name. Return True/False.

3. `async def resolve_owners(meeting_id: str, user_id: str) -> list[OwnerResolutionResult]` --
   - Fetch all action items for this meeting where owner_name is not null and owner_id is null
   - Fetch all people for this user
   - For each action item owner_name:
     a. Check exact match against person.name (case-insensitive) -> resolved, confidence=1.0
     b. Check alias match against person.notes/aliases -> resolved, confidence=0.95
     c. Check fuzzy match (threshold >= 0.8) -> if exactly one match: resolved, confidence=score; if multiple: ambiguous with candidates list
     d. If no match found -> unresolved
   - Update each action_item with owner_id, owner_status
   - Return list of OwnerResolutionResult for logging
