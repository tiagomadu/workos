**supabase/migrations/00002_google_tokens.sql** — New migration for token storage:
```sql
-- Google API Token Storage
CREATE TABLE IF NOT EXISTS user_google_tokens (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES auth.users(id) UNIQUE,
    access_token TEXT NOT NULL,
    refresh_token TEXT NOT NULL,
    expires_at TIMESTAMPTZ NOT NULL,
    scopes TEXT NOT NULL DEFAULT '',
    created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
    updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);

CREATE TRIGGER trg_user_google_tokens_updated_at
    BEFORE UPDATE ON user_google_tokens
    FOR EACH ROW EXECUTE FUNCTION update_updated_at();

ALTER TABLE user_google_tokens ENABLE ROW LEVEL SECURITY;
CREATE POLICY user_google_tokens_policy ON user_google_tokens
    FOR ALL USING (user_id = auth.uid());
```

**backend/app/core/google_oauth.py** — Google OAuth helper:

1. `def get_google_auth_url(redirect_uri: str) -> str`:
   - Build Google OAuth 2.0 authorization URL
   - Scopes: `https://www.googleapis.com/auth/calendar.readonly`, `https://www.googleapis.com/auth/gmail.readonly`
   - Include access_type=offline and prompt=consent for refresh token
   - Return the URL string

2. `async def exchange_code_for_tokens(code: str, redirect_uri: str) -> dict`:
   - POST to https://oauth2.googleapis.com/token with code, client_id, client_secret, redirect_uri, grant_type=authorization_code
   - Use httpx.AsyncClient
   - Return dict with access_token, refresh_token, expires_in, scope
   - Raise HTTPException on failure

3. `async def refresh_access_token(refresh_token: str) -> dict`:
   - POST to https://oauth2.googleapis.com/token with refresh_token, client_id, client_secret, grant_type=refresh_token
   - Return dict with new access_token, expires_in
   - Raise HTTPException on failure

4. `async def get_valid_token(user_id: str) -> str`:
   - Fetch token from user_google_tokens table by user_id
   - If expires_at < now(): call refresh_access_token, update DB, return new token
   - Else: return existing access_token
   - If no token found: raise HTTPException 401 "Google account not connected"

5. `async def store_tokens(user_id: str, tokens: dict) -> None`:
   - Upsert into user_google_tokens: access_token, refresh_token, expires_at (now + expires_in seconds), scopes
   - Use supabase.table("user_google_tokens").upsert() with on_conflict="user_id"

6. `async def revoke_token(user_id: str) -> None`:
   - Delete from user_google_tokens where user_id matches
   - Optionally POST to https://oauth2.googleapis.com/revoke to invalidate with Google

**backend/app/services/calendar_match.py** — Auto-match service:

1. `def compute_match_score(meeting: dict, event: dict) -> tuple[float, list[str]]`:
   - Score based on:
     - Date match: meeting_date matches event date → +0.5 score, reason "Same date"
     - Attendee overlap: compare event attendee names/emails against transcript text → +0.1 per match, reason "Attendee [name] found in transcript"
   - Return (score, list_of_reasons)

2. `async def find_matching_events(meeting_id: str, user_id: str) -> list[dict]`:
   - Fetch meeting record (meeting_date, raw_transcript)
   - If no meeting_date: return []
   - Query calendar_events within ±1 day of meeting_date for this user
   - Score each event using compute_match_score
   - Filter to score >= 0.5 (at minimum date match)
   - Sort by score descending
   - Return top 5 matches as CalendarMatchSuggestion dicts

3. `async def link_meeting_to_event(meeting_id: str, calendar_event_id: str, user_id: str) -> None`:
   - Update meetings.calendar_event_id = calendar_event_id WHERE id = meeting_id AND user_id = user_id
   - Update calendar_events.meeting_id = meeting_id WHERE id = calendar_event_id AND user_id = user_id

4. `async def unlink_meeting_from_event(meeting_id: str, user_id: str) -> None`:
   - Get current calendar_event_id from meeting
   - Set meetings.calendar_event_id = None
   - Set calendar_events.meeting_id = None for the old event

The callback endpoint:
```python
@router.post("/callback")
async def google_oauth_callback(
    body: GoogleCallbackRequest,  # { code: str }
    user: AuthenticatedUser = Depends(get_current_user),
):
    tokens = await exchange_code_for_tokens(body.code, settings.GOOGLE_REDIRECT_URI)
    await store_tokens(user.id, tokens)
    return {"status": "connected"}
```
